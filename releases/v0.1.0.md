**xk6-tcp** `v0.1.0` is here 🎉!

This is the initial public release. As this is the initial release, the API may evolve based on community feedback and real-world usage patterns.

**xk6-tcp** is a k6 extension that adds first-class support for raw TCP socket communication to your load testing and performance scripts. With this extension, you can establish TCP connections, send and receive data, and test network protocols directly from your k6 tests.

The [xk6-tcp API](https://github.com/grafana/xk6-tcp) is intentionally designed to feel familiar to users of Node.js's [`net.Socket`](https://nodejs.org/api/net.html#class-netsocket) API. This means you can leverage event-driven programming, both synchronous and asynchronous operations, and comprehensive lifecycle management. The extension provides a modern, ergonomic developer experience for TCP-based protocol testing in JavaScript.

**xk6-tcp** is production-ready and suitable for testing a wide variety of TCP-based protocols including databases, message queues, custom protocols, and more.

## Features

### Event-Driven Socket API
Modern event-based programming with familiar Node.js-style methods:
- `connect` / `connectAsync` - Establish TCP connections
- `write` / `writeAsync` - Send data with various encodings
- Event handlers for `connect`, `data`, `close`, `error`, and `timeout`
- Full lifecycle management with `destroy()`

### TLS/SSL Support
Secure TCP connections with built-in TLS/SSL encryption:
- Enable with simple `tls: true` option
- Leverages k6's standard TLS configuration
- Support for secure databases, HTTPS-like protocols, SMTPS, IMAPS, and custom secure protocols
- HTTP/1.1 enforcement to avoid HTTP/2 binary frames for better testing experience

### Async/Await Support
First-class async programming support:
- Promise-based `connectAsync()` and `writeAsync()` methods
- Compatible with standard JavaScript async constructs
- Mix and match event-driven and Promise-based patterns

### Comprehensive Metrics
Built-in metrics for performance monitoring:
- `tcp_socket_connecting` - Connection establishment time
- `tcp_socket_resolving` - DNS resolution time
- `tcp_socket_duration` - Total connection duration
- `tcp_sockets` - Connection counter
- `tcp_reads` / `tcp_writes` - I/O operation counters
- `tcp_errors` - Error tracking
- All metrics support custom tags for filtering and aggregation

### Binary Data Support
Handle binary protocols with ease:
- Send and receive binary data (ArrayBuffer)
- Multiple encoding support (utf8, base64, hex, etc.)
- Zero-copy buffer pooling for performance

### Timeout Management
Fine-grained control over socket timeouts:
- Configurable read timeouts with `setTimeout()`
- Timeout events for custom handling
- Automatic cleanup on timeout

### Connection Information
Access detailed connection information:
- Local and remote IP addresses
- Local and remote ports
- Connection state tracking
- Bytes read/written counters

## Quick Start

Here's a simple example demonstrating the basic usage:

```javascript
import { Socket } from "k6/x/tcp";

export default async function () {
  const socket = new Socket();

  socket.on("connect", () => {
    console.log("Connected to server");
    socket.write("Hello, TCP!\n");
  });

  socket.on("data", (data) => {
    const response = String.fromCharCode.apply(null, new Uint8Array(data));
    console.log("Received:", response);
    socket.destroy();
  });

  socket.on("close", () => {
    console.log("Connection closed");
  });

  socket.on("error", (err) => {
    console.error("Socket error:", err);
  });

  await socket.connectAsync({
    port: 8080,
    host: "localhost"
  });
}
```

## TLS/SSL Example

Testing secure connections is straightforward:

```javascript
import { Socket } from "k6/x/tcp";

export default async function () {
  const socket = new Socket();

  socket.on("connect", () => {
    console.log("TLS connection established");
    socket.write("GET / HTTP/1.1\r\nHost: example.com\r\n\r\n");
  });

  socket.on("data", (data) => {
    const response = String.fromCharCode.apply(null, new Uint8Array(data));
    console.log("Response:", response.substring(0, 100));
    socket.destroy();
  });

  await socket.connectAsync({
    port: 443,
    host: "example.com",
    tls: true  // Enable TLS encryption
  });
}
```

## Installation

```bash
xk6 build --with github.com/grafana/xk6-tcp@v0.1.0
```
